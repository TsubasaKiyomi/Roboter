##### roboko の定型文

```
- 「こんにちわ,私の名前は roboko です.あなたの名前は？」

- 「私のおすすめレストランは + レストラン_name + です。このレストランはお好きですか？」

- 「user さんのおすすめレストランはどこですか？」

- 「userさん。ありがとうございます。良い一日を。」

```

```
- おすすめレストランを聞かれた際の解答できるフォーマット
```
